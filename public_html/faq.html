FAQ

What is this site?

A dynamic map of all hacker/maker spaces and fablabs. There is already some map out there, but you have to register. After a while the map wil be outdated, so for this map there are several 'live' souces used to be always uptodate.

What sources are used?

There are tree main sources of data :
  • spaceapi directory
    read status via json
  • fablab.oi labs list
    Fablab should have status 'active'
  • hackerspace.org symantic data
    Extra checks if already done bij spaceapi, more then 1 members and if entry older then 3 years if site still up.

Can i help you?

Cool, thats nice of you. You could help me several ways :
  • Pay a small donation to cover cost of hosting etc.
  • If an entry is invalid (e.g. hackerspace close) edit this at the source.

Can you update my entry?

I don't keep a database of your data, I update this every day from the mentioned datasources. If you want to see where I got your data from, click on your icon and select 'source'. That should bring you to the data source where you can change your data. Allow at least 24 hours to update on this map.
